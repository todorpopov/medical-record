This project has been built as the final project to the CSCB869 Java Web Services course at New Bulgarian University by Todor Popov - F112420.

The system follows the microservices architecture, utilizing multiple Java/Maven services and an Angular frontend.

The frontend is served to the client's browser via Nginx.

It uses a Spring Boot RESTful API to act as a gateway between the services and the frontend. Inter-service communication is handled synchronously via gRPC, and asynchronously via RabbitMQ message queues.

The system uses a shared PostgreSQL instance to persist data inside multiple, separate databases for individual microservices.

All persistence operations are handled by the Hibernate ORM.

All of the individual services and dependencies run inside Docker containers, and are orchestrated using Kubernetes.
